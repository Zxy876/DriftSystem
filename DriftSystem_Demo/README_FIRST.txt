DriftSystem Demo 使用指南（給老師）

這個 Demo 主要展示 DriftSystem 的故事與情境設計。請注意：這不是 Minecraft 客戶端，仍需自備 Minecraft Java 版才能進入世界。

為什麼推薦 HMCL 啟動器？
· 不需要綁定帳號即可直接體驗，適合課堂展示與教學。
· 操作介面簡單，安裝與啟動流程清楚。
· 下載來源官方可信，網址：https://hmcl.huangyuhui.net/

完整體驗流程（依序完成）：
Step 1：下載 HMCL（建議放在容易找到的位置），開啟後按照畫面提示完成安裝。
Step 2：在 HMCL 中選擇「遊戲版本」，安裝 Minecraft Java 版 1.20.1（這個版本與 Demo 世界相容）。
Step 3：下載 DriftSystem_Demo.zip，將整個資料夾解壓到桌面或其他方便的位置。
Step 4：打開解壓後的資料夾，雙擊「启动游戏.bat」，系統會自動準備所需內容，第一次啟動請耐心等待。
Step 5：保持兩個黑色視窗開著，回到 HMCL，啟動 Minecraft，進入「多人遊戲」，新增伺服器位址「localhost」後加入，即可開始體驗。

重要提醒：
· 第一次啟動需要一些時間，因為系統會自動安裝必要的套件，請不要關閉視窗。
· 體驗過程中務必保持那兩個黑色視窗開啟，關閉視窗會中斷 Demo。
· 這是藝術研究用的展示內容，並非商業遊戲產品。

祝操作順利，歡迎在課堂或展覽中使用這個 Demo 展示 DriftSystem 的想像力與故事性。
