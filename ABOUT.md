# 关于 DriftSystem · 心悦宇宙

DriftSystem 是一个实验性的 AI 驱动游戏项目，探索自然语言与虚拟世界的交互边界。

## 项目愿景

我们相信未来的游戏体验应该是：
- **对话驱动** - 用自然语言代替复杂的命令系统
- **AI 增强** - 实时生成剧情、关卡和世界事件
- **社交协作** - 多人共同塑造游戏世界

## 技术栈

- **前端**: Minecraft Paper 1.20.1 + Java 17
- **后端**: Python FastAPI + DeepSeek AI
- **通信**: HTTP/JSON API
- **数据**: JSON 格式的关卡 DSL
