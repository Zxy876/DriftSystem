# DriftSystem Resource Pack

This directory stores placeholder assets for DriftSystem NPC skins.

## Structure

- `pack.mcmeta`: Declares the resource pack for Minecraft 1.20.x (update `pack_format` if Mojang changes it).
- `assets/minecraft/textures/skins/`: Drop finalized NPC skin textures here.

## Notes

- Keep placeholder files under version control using `.gitkeep`.
- Do not reference these assets in code until production-ready textures exist.
- When targeting a specific 1.20 sub-version, confirm the correct `pack_format` value via the official Minecraft documentation.
